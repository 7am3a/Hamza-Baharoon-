# Hamza-Baharoon-
Pertama University is establishing a new campus in Johor Bahru and requires a secure, high-availability network infrastructure that supports diverse user roles, secure wired/wireless connectivity, and remote access to sensitive systems at the main campus (at Cyberjaya) data center.
Task: 
Designing secure and scalable network architecture.
Implementing and simulating key configurations.
Ensuring secure inter-campus communication (from new campus to Main Campus at Cyberjaya)
Proposing access control, segmentation, and secure service hosting strategies
Roles:
Network Security Analyst: Network Security Analyst Risk assessment, threat modeling, firewall rules, ACLs, IPS, SIEM.
Network Architect: Network Architect VLAN design, subnetting, routing configuration, inter-campus connectivity.
Systems & VPN Engineer:Secure access to data center (IPSec/SSL VPN), server security hardening.
Wireless & NAC Specialist: Configure secure wireless (WPA3/802.1X), NAC using RADIUS/LDAP
